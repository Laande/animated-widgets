# Animated Widgets

An Android application that allows you to create home screen widgets from images and animated GIFs.

## Features

- Create widgets from any image or GIF
- Animated GIF support
- Toggle animation on/off for each widget
- Rename and manage your widgets

## Requirements

- Android 8.0 (API 26) or higher
- Storage permission for selecting images

## Installation

Download the latest APK from the [Releases](https://github.com/laande/animated-widgets/releases) page.
